  • table 表示表格
    • border 属性用于设置 表格的边框 的 粗细
    • bordercolor 属性用于设置表格边框的颜色
    • align 属性用于控制表格本身的位置 ( 水平排列方式 : left 、center 、right )
    • width 属性用于控制表格的宽度 ( 可以使用 百分比 或 像素 )
    • bgcolor 属性用于设置 整个表格 的 背景颜色
  • tr : table row ，表格行
    • bgcolor 属性用于设置 表格行中的每个单元格的 背景颜色
  • td : table data cell ，表示表格行中的单元格
    • 当 table 元素的 border 不为 0 时单元格的边框也会显示
    • 当 table 元素的 bordercolor 属性指定了表格边框的颜色时，单元格的边框颜色也与其相同
    • width 属性用于控制单元格的宽度 ( 可以使用 百分比 或 像素 )
    • height 属性用于控制单元格的高度 ( 可以使用 百分比 或 像素 ) ( 同一行以最高的为准 )
    • bgcolor 属性用于设置 单个单元格的 背景颜色
